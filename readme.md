# 🚚 ReliefLink AI

### Intelligent Crisis & Disaster Logistics System

## 📌 Overview

ReliefLink AI is a smart logistics system designed to optimize the distribution of essential resources during crisis and disaster situations.

## 🚀 Features

* Priority-based resource allocation
* Demand vs supply analysis
* Optimized delivery routing
* Real-time decision support (prototype)

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

## 🎯 Objective

To ensure efficient and timely distribution of resources by transforming crisis response into a data-driven decision-making process.
